# 貢獻指南

感謝你有興趣貢獻！

## 如何貢獻

1. Fork 本項目
2. 創建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add AmazingFeature'`)
4. 推送分支 (`git push origin feature/AmazingFeature`)
5. 創建 Pull Request

## 代碼規範

- 使用 ShellCheck 檢查腳本
- 保持簡單，不要過度複雜
- 添加註釋說明複雜邏輯

## 問題回報

請使用 GitHub Issues，包含：
- 你的環境 (OS, OpenClaw 版本)
- 問題重現步驟
- 預期 vs 實際行為
